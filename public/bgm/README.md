# BGM配置ガイド

各画面のBGMは下のフォルダに入れて差し替える想定です。

- `title/`
  - タイトル画面用
  - 推奨: 明るい・期待感・ゆっくり立ち上がる
  - 目安: 90-115 BPM / ポップ / 8bit / 軽いエレクトロ
  - 候補名: `title_main.mp3`, `title_alt.mp3`
- `host/`
  - ホスト設定画面用
  - 推奨: 落ち着いて選びやすい・邪魔しない
  - 目安: 80-105 BPM / ローファイ / チル / 軽いジャズ
  - 候補名: `host_lobby.mp3`, `host_setup.mp3`
- `play/`
  - プレイ中の共通BGM用
  - 推奨: 集中しやすい・テンポ感あり・長時間でも疲れにくい
  - 目安: 105-128 BPM / エレクトロ / スポーツ / リズム強め
  - 候補名: `gameplay_main.mp3`, `gameplay_high_tension.mp3`
- `bomber/`
  - クイズボンバー専用BGM用
  - 推奨: テンポが速い・緊張感・短いループでも飽きにくい
  - 目安: 118-140 BPM / エレクトロ / 8bit / アーケード感
  - 候補名: `bomber_lobby.mp3`, `bomber_battle.mp3`, `bomber_last10.mp3`, `bomber_results.mp3`
- `results/`
  - 結果画面用
  - 推奨: 達成感・表彰感・少し華やか
  - 目安: 90-120 BPM / ファンファーレ寄り / 勝利感
  - 候補名: `results_win.mp3`, `results_summary.mp3`
- `shared/`
  - 汎用SEや画面またぎで使いたい素材用
  - 候補名: `countdown.mp3`, `transition.mp3`

## UI別おすすめ

- タイトル画面
  - 元気で入りやすい
  - 子ども向けでも軽すぎず、ゲーム開始前の期待感を優先
- ホスト画面
  - 長く流れても疲れない
  - 問題選択の視認性を邪魔しない
- プレイヤー画面
  - リズムがありつつ、問題文や効果音を埋もれさせない
  - 低音が強すぎないものが向く
- クイズボンバー
  - 爆弾設置と移動のテンポに合わせて少し速め
  - 爆風SEや正誤音を埋もれさせないよう、中高域中心が向く
  - 候補:
    - `bomber_lobby.mp3`: ルーム待機や開始前の軽い緊張感
    - `bomber_battle.mp3`: 通常バトル用の主旋律
    - `bomber_last10.mp3`: 残り10秒など終盤加速用
    - `bomber_results.mp3`: クイズボンバー専用の勝敗まとめ
- 結果画面
  - 祝福感のある短めループ
  - リトライの邪魔にならない長さ

## 置き方

- ひとまずこのフォルダ構成に音源を入れれば整理できます。
- 再生ファイル名を固定したくなったら、その名前に合わせて `src/lib/sound.ts` を差し替えます。
- クイズボンバー専用BGMを実際に再生させる場合は、`public/bgm/bomber/` に置いたうえで `src/lib/sound.ts` の再生先を追加します。
